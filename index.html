Key Features

Universal Compatibility

Works with any S3-compatible storage service including Minio, Ceph, OVH, Scaleway, Hetzner and more.

Cross-Platform

Native application available for macOS, Windows, and Linux. Small size <20MB without Electron.

Connection Manager

Save and manage multiple S3 service configurations easily.

File Management

Browse, upload, download, and delete files with ease. Generate temporary signed download links.

Search & Filter

Full-text search functionality to find and filter your files quickly.

Folder Tree View

Show virtual folders in a tree view.

Drag & Drop Upload

Upload files by dragging them in to the file list.

Download Links

Generate pre-signed temporary download links.

Progress Tracking

Visual progress tracking for all operations with responsive UI.
