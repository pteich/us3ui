us3ui

Desktop Client for S3 Object Storage

A fast, native application to manage files on your S3-compatible storage. Works with self-hosted instances and managed services across macOS, Windows, and Linux.

Key Features

S3 Compatible

Connects to standard S3 services including MinIO, Ceph, AWS, and providers like Hetzner, OVH, or Scaleway.

Native Performance

Built with Go and Fyne for speed. A single, small executable (under 20MB) that runs natively without the overhead of web technologies.

Multi-Profile Support

Store configurations for multiple buckets or services. Switch between environments instantly. Credentials stay local.

File and Bucket Operations

Standard file management: browse, upload, download, and delete objects. Create and delete buckets.

Quick Search

Filter large buckets instantly with full-text search.

Flexible Viewing

Toggle between a hierarchical folder structure or a flat list of all objects.

Easy Uploads

Drag files from your desktop directly into the application to start uploading.

Shareable Links

Create temporary, pre-signed URLs to share private objects securely.

Operation Status

Clear visual feedback for uploads, downloads, and other interactions.

Installation Notes

The application is distributed as a portable binary and is not currently code-signed. First-time launch often requires manual approval.

macOS

  1. Extract the downloaded archive.
  2. Move the app to your Applications folder.
  3. Right-click the app icon and select Open. Confirm the security prompt.
  4. If the app is blocked, go to System Settings → Privacy & Security and allow the application to run.

Windows

  1. Extract the ZIP file.
  2. Run the executable.
  3. If Microsoft Defender SmartScreen appears, click More info and then Run anyway.
